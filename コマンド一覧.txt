フロントエンド起動
	frontend直下で
  　　  「npx expo start」
	コマンド入力を求められるので
	「a」 (androidなら) 
　　　　起動するまで待機(５分ぐらい)

	一度起動すれば、保存したら自動で反映されると思います。
	＊起動時にエラー出たら npm install してみてください。
バックエンド起動
	バックエンド作り始めたら書きます。

git コマンド

	変更をアップロード
	git add .
	git commit -m "ここにコメント　名前"
	(例、git commit -m "画面遷移のテンプレートを作成　そうし")
	git push
	＊エラーが出たら、git pull してみてください（git pull するとファイルが書き換わるので別のところのコピー作ってね！）

	他の人がアップロードしたコードを取り込む
	git pull

エラーが出るコードをアップロードしないでね！
あと、git に不慣れなんでコード消えてもいいように、git 関係ないところにコピーを保存したほうがいいよ！
